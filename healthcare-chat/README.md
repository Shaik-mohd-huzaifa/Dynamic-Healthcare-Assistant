## Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd healthcare-chat
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:5173`
